1.주석이란 무엇이며, 종류는? 
- 주석이란? 컴파일러 되지 않게 코드에 메모할 수 있는 기능이며,
  작성한 코드를 타인이 혹은 본인이 나중에 쉽게 알 수 있게 정리해 둔 메모.
- 블록단위주석 /* 내용 */
- 행단위주석 // 내용 (주석단축키 ctrl+shift+c )

2.주석은 컴파일시 어떻게 되는가?
?- 주석 처리 된 부분은 제외하고 컴파일러 된다.

3.들여쓰기는 왜 해야 되는가?
- class와 {} 에 대한 소스코드 내용을 의미, 즉 class 코드 안에 속한다는 의미를 지님.
- 코드의 가독성을 높여 이해도를 증가시키는 역할을 한다.

4.변수란 무엇인가?
-Variable, 값을 저장할 수 있는 메모리 상의 공간을 의미하며, 메모리 공간의 할당과 접근을 위해 필요한 도구이다.

5.8형제(자료형)을 써보세요(feat.외우세요).
- 참과 거짓: boolean 1바이트
- 문자: char 2바이트 (아스키문자표, 유니코드문자표) 
  cf) "문자열", '문자'를 넣어야 error 발생하지 않음.
- 정수: byte 1바이트 < short 2바이트 < int 4바이트 < long 8바이트 
  (표현 가능 범위 순으로 작성)
- 실수: float 4바이트 < double 8바이트

6.변수선언의 의미는 무엇인가?
- 변수의 선언은 메모리 공간 할당으로 이어지며, 그와 동시에 변수의 이름과 용도가 결정된다.
- ex) int num1;
      num1 = 10;
변수 num1을 선언하면 num1 이라는 메모리 공간이 생성되고 할당된 공간에 정수 10이
컴퓨터에서 인식할 수 있는 기계어 이진수로 변환되어 입력되는 것!

7.int 형 범위는 어떻게 되는가?
- 1byte = 8bit, int는 4byte이므로 32bit, 
 즉 메모리 상에 2의 32승까지의 할당 공간이 생성되며 범위를 잡게 되면 -2,147,483,648 ~ 2,147,483,647 에 해당

8.int num; 을 메모리로 설명해 보세요.
?- num 이라는 메모리 공간이 생성되고 할당된 공간에 int 라는 정수가 입력됨.

10.아스키 코드란 무엇인가요?
-입력된 문자나 기호들을 컴퓨터가 인식할 수 있도록 아스키 코드표를 이용하여 이진수로 변환.
-아스키코드표(영어기준-표준), 유니코드표(통합)
